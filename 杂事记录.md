- 在内存中分配变量地址空间，是从高地址到低地址分配，还是从低到高分配的？
    - 答：从高往低分配的
####
    #include <stdio.h>
    int main(){
        int var = 20;
        int *p1;
        int **p2;
        p1 = &var;
        p2 = &p1;
        printf("var变量的地址：%x\n", &var);
        printf("p1的值为：%x\n", p1);
        printf("p1的地址：%x\n", &p1);
        printf("p1所指的值：%d\n", *p1);
        printf("p2的值为: %x\n", p2);
        printf("p2的地址：%x\n", &p2);
        printf("p2所指的值：%d\n", **p2);
        return 0;
    }
执行结果如下：

    var变量的地址：428fdb0c
    p1的值为：428fdb0c
    p1的地址：428fdb00
    p1所指的值：20
    p2的值为: 428fdb00
    p2的地址：428fdaf8
    p2所指的值：20
